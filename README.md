# Projeto de um Gerenciador de Hospedagens utilizando Nuxt 4.

## Descrição

Esse projeto surgiu de um projeto de extensão universitária, onde o objetivo era criar um sistema de gerenciamento de hospedagens para uma pequena pousada na cidade que queria maior visibilidade para o seu site e uma forma de gerencias os hóspedes/funcionários do local. O projeto foi desenvolvido inicialmente por uma equipe de 4 pessoas, entretanto, pelo tempo escasso de 4 meses, o projeto não foi finalizado.

Resolvi, após um tempo, então, dar continuidade ao projeto, porém refatorando ele por completo, utilizando tecnologias mais modernas (como o Nuxt 4, Drizzle ORM, Tailwind CSS, entre outras ferramentas) e que eu tinha recém aprendido, com intuido de praticar os conceitos aprendidos.

## Tecnologias Utilizadas

- Nuxt 4
- Nuxt/Ui (Frontend do Nuxt)
- Nitro (Backend do Nuxt)
- Better Auth (Autenticação)
- Drizzle ORM
- Tailwind CSS
- TypeScript
- Node.js
- SQLite
- Turso (banco de dados)
